# TypeGG-Goal-Tracker
Track your goals directly on TypeGG. Create personalized goals to track your typing progress on the modern Typing Website https://typegg.io/ in real time. TypeGG Goal Tracker is a minimalistic, customizable tool to keep you motivated and improve your typing speed.

## GOAL TYPE:
The extension supports 6 goal types: EXP, PP (performance points), Races, Quotes (unique texts), Time, Chars (characters).

## GOAL MODE:
The extension supports 4 goal modes: gain, target, rank, player.
Note: Rank and player mode are available for the EXP and PP goals.

Rank Mode allows you to set a goal of reaching a certain rank on the EXP / PP leaderboards, it tells you how much EXP / PP you need to gain to reach EXP / PP of that rank. The target EXP / PP required is updated automatically since the required PP for a rank can change at any time.

There is a button on rank mode called "Next Rank". If a goal is created while this button is activated, it creates a goal for the next rank. When you reach the goal by increasing your rank, a new goal is automatically created for the next rank.

Player Mode allows you to set a goal of reaching the EXP / PP of a certain player. The target EXP / PP required is updated automatically since the EXP / PP of the target player can change at any time.

## RECURRENCE:
The extension supports recurring goals: daily, weekly, monthly with a streak display

Use the goal presets, or set your own custom goals.

## NOTE:
- Goals are stored in localStorage
- For this extension to work properly you need to have an account
