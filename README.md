TypeGG Goal Tracker allows you to create a variety of highly customizable goals that show your progress with real-time feedback to improve your typing experience on the competitive typing website TypeGG (https://typegg.io/).

Example Goals:
1. Type 10 races per day
2. Reach rank #100 on the leaderboard
3. Overtake a specified player on the leaderboards.
4. Improve by a cumulative 50 WPM across all your quotes.
5. Beat all the scores of a specified player
6. Type at an average speed of 75 WPM for 15 races.
7. Reach the next rank on the leaderboards

and many more...

Information about the Extension:
- To use the extension you need to have an account.
- TypeGG Goal Tracker uses the official TypeGG public API (https://api.typegg.io/v1/docs) to retrieve various stats for the goals.
- Your data is stored locally in your browser (localStorage)
- You can backup your data via the settings.
